## Назначение проекта

Учебный PET-проект, для понимания устройства работы FastAPI, REST, pytest, а так же, для дополнительной практики в Docker, PostgreSQL, SQLAlchemy.
Возможно будет реализован фронтенд.

## Что делает проект

Лента новостей, с доступом только для авторизованых пользователей. 
 - Получает из микросервисов (возможно будут лежать в данном репозитории) посты с краткой выжимкой новостей за предыдущий или текущий день.
 - Сохраняет посты (только текстовая информация) в БД.
 - При запросах может показать как ленту новостей, так и пост за текущий день.
 - Можно выбрать пост за предыдущие дни.
 - Возможно будет делать два поста в день, утром и вечером.
 - Возможность добавлять новых пользователей.

## Стек технологий:

FastAPI, Pydantic, pytest-asycnio, RESTapi, PostgreSQL, SQLAlchemy, SQLModel, Docker, Redis
